在Win上运行 Apple Music 下载器 整合包版

中国大陆用户须知

  1. 确保使用稳定的网络环境和有效的代理工具（推荐开启 TUN 模式）
  2. 按 Win + R 输入 cmd 打开命令提示符
  3. 执行以下命令验证网络状态：
  4. 出现连续响应且无丢包（如图示）即为正常

一、环境部署

步骤 1：基础安装

  1. 下载安装包：
    AMDL-WSL1 整合包
  2. 解压后按顺序执行：
    • 0-1-install-dependencies.bat → 如需重启请完成重启
    • 0-2-configure-wsl.bat

二、下载流程

步骤 1：配置解密器

  1. 用文本编辑器打开 1-start-wrapper.bat
  2. 修改以下字段：
  3. 保存后运行此批处理文件

步骤 2：登录验证

  • ✅ 成功标志：出现 type6 提示和登录信息
  • ❌ 失败处理：
    • 出现 _bionic_open_tzdata_path：检查网络连接
    • 需要二次验证：在窗口输入验证码
  • ➡️ 保持此窗口开启状态

步骤 3：执行下载

  1. 运行 2-start-downloader.bat 开启新终端
  2. 根据需求选择命令：
下载类型 命令格式 示例
整张专辑/歌单 go run main.go [URL] go run main.go https://music.apple.com/cn/album/...
杜比全景声专辑 go run main.go --atmos [URL] go run main.go --atmos https://...
选择性下载曲目 go run main.go --select [URL] 输入 1-3,5,6 选择曲目

三、高级配置

修改存储路径

  2. 修改路径映射示例：

获取高级功能权限

要解锁歌词/AAC-LC/MV下载功能：

  1. 获取有效的 media-user-token
  2. 确保 token 与下载区域账号匹配
  3. 打开 Apple Music 并登录
  4. 打开开发者工具，点击“应用程序 -> 存储 -> Cookies -> https://music.apple.com”
  5. 找到名为“media-user-token”的 Cookie 并复制其值
  6. 将步骤 3 中获取的 Cookie 值粘贴到 config.yaml 文件中并保存
  • 正常启动脚本
